# 🚀 CodeLens AI

CodeLens AI is an AI-powered code review platform that analyzes source code, detects bugs, suggests improvements, and recommends best coding practices using Google's Gemini API.

---

## ✨ Features

- 🤖 AI-powered code review
- 📝 Syntax-highlighted code editor
- 💡 Intelligent suggestions and optimizations
- ⚡ Real-time code analysis
- 🔒 Secure API key management using environment variables

---

## 🛠️ Tech Stack

### Frontend
- React
- Vite
- Axios
- PrismJS
- React Simple Code Editor
- React Markdown

### Backend
- Node.js
- Express.js
- Google Gemini API
